## My super album app

This app allows you to keep a database of existing albums in your collection

### Example set up

1. Clone the repo
1. Add your api key.

   **Example:**

   ```
   API_SECRET="w1alrXUtnJ3fI_A7BER+Sx0NxR3f3R3NadawCku3P12K3d"
   ```
   
   ```
   PASSWORD="some_super_strong_passw0rd!"
   ```

1. Run the application.
